# Expense Tracker 💎

A premium dark-themed expense tracker web app built with React.

## Features

- **Authentication** - Sign up and log in with persistent sessions
- **Add Expenses** - Name, amount, category (Food, Travel, Bills, Other), and date
- **View & Filter** - See all expenses, filter by category, view category-wise totals
- **Delete** - Remove any expense with a smooth animation
- **Persistent Storage** - Data saved across sessions

## Tech

- React (functional components + hooks)
- Persistent key-value storage API
- INR currency formatting
- Dark luxury UI with gold accents
